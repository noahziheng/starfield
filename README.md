# 宁静星空 (Starfield)

一个宁静的交互式星空效果，跟随鼠标手势移动。

## 功能特点

- 🌟 **500颗闪烁星星** - 每颗星星都有独特的大小、亮度和闪烁频率
- 🌌 **视差效果** - 星星根据深度产生不同程度的视差移动
- ☄️ **随机流星** - 偶尔划过天际的流星效果
- 🌈 **星云背景** - 柔和的星云增添氛围
- 🖱️ **鼠标交互** - 移动鼠标探索星空
- 📱 **触摸支持** - 支持移动设备触摸操作

## 使用方法

直接在浏览器中打开 `index.html` 文件即可体验。

```bash
# 或者使用本地服务器
npx serve .
# 然后访问 http://localhost:3000
```

## 文件结构

```
starfield/
├── index.html    # HTML 结构
├── styles.css    # 样式文件
├── script.js     # 交互逻辑
└── README.md     # 说明文档
```

## 技术实现

- **Canvas 2D** - 使用 HTML5 Canvas 绘制星空
- **requestAnimationFrame** - 流畅的60fps动画
- **视差滚动** - 基于深度的视差效果
- **平滑插值** - 鼠标移动的平滑过渡

## 自定义

可以在 `script.js` 中调整以下参数：

- `starCount` - 星星数量（默认500）
- `nebulaCount` - 星云数量（默认5）
- `maxShootingStars` - 最大流星数量（默认3）
- 流星触发概率（默认0.5%每帧）
